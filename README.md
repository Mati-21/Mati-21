# 💫 About Me

I am **Mati Melkamu**, a **Software Engineering student at Arba Minch University** and a passionate **Full-Stack Developer** focused on building modern, scalable, and interactive web applications.

My core expertise lies in the **MERN stack**, where I build full-stack applications using **React, Node.js, Express, and MongoDB**. I also develop **real-time applications using Socket.io**, enabling interactive multiplayer and live systems.

On the **frontend**, I focus on building responsive and performant user interfaces using **React, Tailwind CSS, Sass, Redux, and TypeScript**.

On the **backend**, I develop scalable REST APIs using **Node.js and Express**, and work with databases including **MongoDB, PostgreSQL, and MySQL**.

I also have hands-on experience with **Docker for containerization**, **Postman for API testing**, **Git for version control**, and **Figma for UI design**.

Currently, I am focused on **building real-time systems and large-scale web applications**, while continuing to expand my expertise in **full-stack development and cloud technologies**.
